# Portfólio Pessoal 

Este é o repositório do meu portfólio pessoal, desenvolvido para apresentar minhas habilidades, projetos e minha trajetória na área de desenvolvimento de software. O foco deste projeto foi construir uma interface limpa, rápida e responsiva utilizando apenas HTML, CSS e Javascript sem a dependência de frameworks externos.

## Tecnologias Utilizadas

O projeto foi construído utilizando:

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização, layout responsivo (adaptável para dispositivos móveis e desktops).
*   **Javascript:** Por enquanto utilizado pontualmente a atualização automática do ano no rodapé. 

## Funcionalidades

*   Apresentação profissional e currículo.
*   Seção dedicada à exibição de projetos técnicos.
*   Design responsivo.
